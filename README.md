# Superstore Sales - Advanced Exploratory Data Analysis

## Overview
An in-depth Exploratory Data Analysis (EDA) on the Superstore Sales dataset 
uncovering 10+ actionable business insights using Python, Pandas, Matplotlib and Seaborn.

## Dataset
- 10,191 rows | 21 columns
- Source: Kaggle - Superstore Sales Dataset

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Business Insights
1. Technology category has highest sales AND highest profit margin
2. West region is most profitable
3. Standard Class is most preferred shipping mode
4. Sales peak in November-December every year
5. Binders have highest average discount
6. Sean Miller is top customer by sales
7. Discount negatively impacts profit (-0.22 correlation)
8. Consumer segment drives maximum sales
9. Year-wise sales show consistent growth (2023-2026)
10. Tables and Bookcases are least profitable sub-categories
